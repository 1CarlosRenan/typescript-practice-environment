# Instalação

Requisito é já ter o npm para a instalação do TypeScript. Instalando globalmente:

```
npm i -g typescript
```

Verificando a instalação a partir da versão:

```
tsc -v
```

# Comandos Gerais

```
tsc --init
```

```
tsc
```

```
tsc -w
```
